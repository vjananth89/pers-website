A PSL Search Engine for Twitter   Python Cassandra Titan Elasticsearch Java
May '14 - May '15

"PSL Search Engine for Twitter" is a project that was conceived with a vision of going beyond the basic keyword search functionalities and using Probabilistic Soft Logic, a machine learning method to gather related tweets in the form graphs. Graphs typically contain related tweets based on location, time, and several other factors. The Graphs can be stored as GraphSON data on Titan which is built on top of Cassandra. The use of a distributed search engine such as Elasticsearch to search the stored data makes the project and exciting prospect.

My Responsibilities
  • Storing the processed tweets data into Titan and Cassandra.
  • Setting up clusters in Cassandra and the creation of a distributed architecture for the project.
  • Collaborating with front end engineers to help integrate Elasticsearch with Java and create a UI page for the project.
  • Explore the use of Javascript plugins with Elasticsearch to help visualize the graphs and tweets.
  • Professor
  • Dr. Feng Chen

UAlbany Market   Java  MySQL  JSP  Selenium  Amazon S3  Amazon EC2 
Jan '13 - May '13

The aim of creating UAlbany Market was to build a shopping website that would mimic all the features of a well established website like Amazon. The website was built using Spring MVC architecture. Twitter Bootstrap, JSP, and HTML/CSS were utilized to create the front end. MYSQL database was used to store buyer, seller, product information. Images of products were hosted in Amazon S3. Selenium was used to perform testing. Payments were set up using PayPal Sandbox. The entire website was hosted on Amazon EC2.
